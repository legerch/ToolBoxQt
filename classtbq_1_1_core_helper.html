static

Use to quit application.

To properly quit application, we have to queue "quit" event, otherwise nothing will be performed (like when quit() is called from a constructor for example).
This method was implemented to properly take care of those edge cases : so it can be called anywhere, quit event will be made.

Parameters
[in]reasonReason for the application to quit.
This will appear in a QMessageBox and be logged.
[in]parentParent widget.
This value can be nullptr.
